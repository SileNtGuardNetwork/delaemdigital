# Усиление AI-стека

Статус: v2 foundation
Назначение: настроить GPT Business и Cursor Pro+ как единую производственную систему для фабрики сайтов.

## 1. Главная позиция

Производственный стек Делаем Диджитал:

```text
GPT Business -> центр управления фабрикой
Cursor Pro+ -> инженерная реализация
```

Claude полностью исключён из производственного стека.

Качество сайтов должно держаться не на вдохновении, а на системе:

```text
стандарты -> агенты -> конверсионный прототип -> дизайн -> код -> интеграции -> QA -> улучшение
```

## 2. Роль GPT Business

GPT Business — центр управления фабрикой.

GPT отвечает за:

- стратегию продукта;
- архитектуру фабрики;
- анализ ниши;
- диагностику бизнеса;
- оффер;
- конверсионный прототип;
- copy-doc;
- ИИ-автоворонку;
- Telegram-сценарии;
- лид-магниты;
- трипвайеры;
- SEO-карту;
- требования к premium-дизайну;
- постановку задач Cursor;
- ревью дизайна;
- ревью кода на уровне архитектуры;
- QA Gate;
- финальную приёмку.

GPT не делает production-деплой, не трогает секреты и не принимает слабые решения без фиксации рисков.

## 3. Роль Cursor Pro+

Cursor Pro+ — инженерная производственная среда.

Cursor отвечает за:

- Next.js;
- TypeScript;
- Tailwind;
- компонентную систему;
- premium-вёрстку;
- адаптив;
- формы;
- server routes;
- Supabase;
- Telegram API;
- оплаты;
- analytics adapters;
- SEO metadata;
- legal pages;
- исправление ошибок;
- рефакторинг;
- локальные проверки.

Cursor не должен:

- менять оффер без команды;
- придумывать структуру сайта с нуля;
- заменять premium-дизайн generic-блоками;
- добавлять fake dashboards;
- трогать secrets;
- коммитить env-файлы;
- запускать production deploy;
- считать задачу готовой без отчёта.

## 8. Критерий сильной конфигурации

Конфигурация считается сильной, если:

1. GPT принимает все продуктовые решения.
2. Cursor не пишет код без прототипа.
3. Cursor не меняет оффер и структуру самовольно.
4. Дизайн проходит отдельный premium Gate.
5. Любой сайт проходит проверку конверсии, SEO, mobile, speed, legal, аналитики, формы и ИИ-воронки.
6. Ни один сайт не запускается без QA Gate.

## 9. Главный акцент

Особый приоритет фабрики — дизайн.

Каждый сайт должен выглядеть современно, дорого, удобно и технологично.

Запрещено выпускать сайты, которые:

- выглядят шаблонно;
- похожи на generic SaaS landing;
- имеют слабый first screen;
- перегружены карточками;
- имеют дешёвый motion;
- плохо читаются на mobile;
- тормозят;
- не ведут пользователя к действию.

- словарь и нейминг фабрики;
