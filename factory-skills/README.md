# Factory Skills — Делаем Диджитал

Статус: v1 registry
Назначение: реестр будущих навыков фабрики сайтов.

## 1. Что такое factory skill

Factory skill — это повторяемая инструкция для конкретного производственного действия фабрики.

Формат ориентируется на Agent Skills / `SKILL.md`, но адаптирован под Делаем Диджитал.

## 2. Принцип

Каждый skill должен отвечать на вопросы:

```text
когда использовать
что получает на вход
что делает
что отдаёт на выход
какие ограничения
какие QA-критерии
```

## 3. Базовые skills

```text
conversion-prototype
premium-hero
expert-photo-system
hero-motion-video
seo-traffic-audit
lead-layer-qa
ai-funnel-architecture
launch-qa-gate
```
