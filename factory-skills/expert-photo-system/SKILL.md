# Expert Photo System Skill

## Когда использовать

Использовать, когда сайт строится вокруг эксперта, владельца, консультанта или личного бренда.

## Вход

- архив фото;
- ниша;
- позиционирование эксперта;
- hero copy;
- дизайн-направление;
- требования mobile;
- CTA map.

## Процесс

1. Разделить фото на категории:
   - hero;
   - portraits;
   - work;
   - details;
   - mobile;
   - og.
2. Выбрать главный hero-кадр.
3. Определить композицию: left/right/center.
4. Проверить safe area под текст и CTA.
5. Определить mobile crop.
6. Выбрать secondary photos для секций.
7. Определить OG image.
8. Зафиксировать photo QA.

## Выход

```text
photo asset map
main hero photo
mobile hero photo
section photo map
OG image decision
photo QA notes
```

## QA

Фото-интеграция готова, если:

- лицо не искажено;
- CTA не перекрыт;
- есть mobile crop;
- фото усиливает доверие;
- нет stock people;
- фон не конфликтует с текстом;
- есть fallback.

## Запреты

- случайный AI-персонаж вместо эксперта;
- фото без обработки и общей стилистики;
- hero без mobile-версии;
- важный текст поверх шумного фона без overlay.
