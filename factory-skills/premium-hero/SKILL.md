# Premium Hero Skill

## Когда использовать

Использовать для проектирования первого экрана сайта перед реализацией.

## Вход

- conversion blueprint;
- hero copy;
- тип сайта;
- аудитория;
- фото / видео / motion assets;
- дизайн-направление;
- primary CTA;
- secondary CTA.

## Процесс

1. Определить тип hero:
   - Founder Portrait Hero;
   - System Map Hero;
   - Diagnostic Hero;
   - Video Hero;
   - Hybrid Motion Hero.
2. Проверить H1: понятен ли за 5 секунд.
3. Выбрать композицию: left text / right visual или наоборот.
4. Определить safe area для текста.
5. Расставить CTA.
6. Добавить trust bullets.
7. Определить mobile layout.
8. Определить motion/fallback.

## Выход

```text
hero type
composition
visual direction
CTA behavior
mobile rules
motion notes
QA checklist
```

## QA

Hero готов, если:

- H1 виден и понятен;
- CTA заметен;
- визуал не мешает тексту;
- mobile не ломается;
- нет raster text;
- нет лишнего декора;
- hero выглядит premium, а не generic.

## Запреты

- H1 внутри изображения или видео;
- CTA внутри изображения или видео;
- floating cards без смысла;
- перегруженный motion;
- stock people для founder-led сайта.
